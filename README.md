# Project
BarberShop.

> Technologies used
    - Node-js, Express-js
    - Supabase

## Installation process
1. #### clone the repo using this command
    git clone https://github.com/zeeshanriasat302/BarberShop.git

2. #### navigate to the project directory
    cd express-supabase

3. #### install npm packages
    npm install

4. #### start the server
    npm start
